How the “ML” works (local)

This page uses a tiny TF-IDF + token matching ranker to select the most relevant FAQ entry. It’s ML-inspired (vector similarity), but runs fully in your browser.

  • Questions are broken into tokens.
  • Each FAQ has a simple TF-IDF vector.
  • User input is vectorized and scored by cosine similarity.
  • Top match becomes the answer template.
